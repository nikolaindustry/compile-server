DELETE this file and this entire hyperwisor-iot folder.

Instead, copy the ENTIRE contents of your local Arduino libraries folder here:

  Your local path (Windows):
    C:\Users\<YourName>\Documents\Arduino\libraries\

  Paste everything INTO:
    compile-server\libraries\

  Final structure should look like:
    compile-server\libraries\
        hyperwisor-iot\         ← your main library
        ArduinoJson\            ← its dependency
        nikolaindustry-realtime\← its dependency
        WiFiManager\            ← any other deps
        ... (all other libraries your .ino uses)

  All of these get copied into /root/Arduino/libraries/ inside Docker,
  which is exactly where arduino-cli looks for them during compilation.

  No need to install them with arduino-cli lib install — just copy the folders.
